# Latin Wiki

Simple english to latin and vice versa translator

## Usage

### 1. Clone this repository


### 2. Install dependencies

Go to the downloaded repository folder and run:
```
npm install
```

### 3. Run the app

```
npm run serve
```

App will be opened in browser at `http://localhost:8080/`

## Move to device

```
cordova run android
```
```
cordova run ios
```
